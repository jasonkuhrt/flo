# flo list worktrees

```
Usage: flo worktree <subcommand>

Subcommands:
  create <name>    Create a new worktree
  delete <name>    Delete a worktree
  list             List all worktrees
  switch <name>    Switch to a worktree

Examples:
  flo worktree create feature-x
  flo worktree delete feature-x
  flo worktree list
```
