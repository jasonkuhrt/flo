# flo list prs

```
Usage: flo list prs [options]

List pull requests for the current repository.

Options:
  --all        Show all PRs (open, closed, merged)
  --closed     Show only closed/merged PRs
  --limit N    Limit number of PRs (default: 20)

Examples:
  flo list prs
  flo list prs --closed
  flo list prs --all --limit 50
```
